# 訪客模式功能說明

## ✨ 新增功能

在現代化 UI 版本的基礎上，新增了**訪客模式**功能，讓使用者無需註冊帳號即可快速使用卡面收集器。

## 🎯 功能特點

### 1. 快速開始
- 點擊「訪客模式」按鈕即可立即使用
- 不需要輸入任何帳號資訊
- 適合想要快速體驗的新用戶

### 2. 本機存儲
- 所有收集數據保存在瀏覽器的 localStorage
- 關閉頁面後重新開啟，數據依然存在
- 存檔按鈕會將數據保存到本機（不上傳雲端）

### 3. 隱私保護
- 不會將任何數據上傳到雲端伺服器
- 所有操作都在本機完成
- 適合注重隱私的使用者

### 4. 功能完整
- ✅ 收集卡片
- ✅ 查看統計
- ✅ 篩選排序
- ✅ 設定背景
- ✅ 本機保存
- ✅ 下載 JSON 檔案
- ❌ 雲端同步（僅帳號模式支援）

## 🖥️ 使用方式

### 方式一：訪客模式
```
1. 開啟 modern.html
2. 點擊「訪客模式」按鈕
3. 開始收集卡片
4. 點擊「存檔」保存到本機
```

### 方式二：帳號模式
```
1. 開啟 modern.html
2. 輸入帳號（英數字組合）
3. 點擊「開始使用」
4. 支援雲端同步
```

## 🔄 模式轉換

### 從訪客模式匯出到帳號模式
```
1. 在訪客模式下點擊「下載」按鈕
2. 將 collected.json 儲存到電腦
3. 重新整理頁面，使用帳號登入
4. 選擇「載入本機檔案」並選擇剛才的 JSON
5. 點擊「存檔」上傳到雲端
```

## 💾 數據存儲位置

### 訪客模式
- **收藏數據**: `localStorage.NeveGuestData`
- **背景設定**: `localStorage.NeveGuestBg`
- **存儲格式**: JSON 字串

### 帳號模式
- **收藏數據**: Google Forms 雲端（加密）
- **本機快取**: `localStorage.NeveUserName`

## ⚠️ 注意事項

### 訪客模式
1. **數據安全**：清除瀏覽器數據會導致收藏記錄丟失
2. **定期備份**：建議定期使用「下載」功能備份數據
3. **單一裝置**：無法跨裝置同步，每個瀏覽器獨立存儲
4. **隱私模式**：在瀏覽器隱私模式下可能無法保存

### 帳號模式
1. **雲端同步**：需要網路連線才能存取雲端數據
2. **跨裝置**：可在不同裝置使用相同帳號存取數據
3. **數據安全**：使用 AES 加密後上傳

## 🛠️ 技術實現

### 核心變數
```javascript
var isGuestMode = false;  // 訪客模式標記
```

### 存檔邏輯
```javascript
function SaveAsFile() {
    if (isGuestMode) {
        // 訪客模式：保存到 localStorage
        localStorage.setItem("NeveGuestData", JSON.stringify(collected));
        localStorage.setItem("NeveGuestBg", JSON.stringify(bg_list));
        alert("已保存到本機（訪客模式）");
    } else {
        // 帳號模式：上傳到雲端
        $.ajax({ /* ... */ });
    }
}
```

### 載入邏輯
```javascript
$("#guestBtn").click(function(){
    isGuestMode = true;
    
    // 嘗試載入訪客數據
    const savedData = localStorage.getItem("NeveGuestData");
    if (savedData) {
        collected = JSON.parse(savedData);
    }
    
    // 進入應用
    $("#loginSection").hide();
    $("#appSection").show();
});
```

## 🎯 適用場景

### 適合使用訪客模式
- 🎮 想要快速試用的新用戶
- 🔒 注重隱私不想註冊的用戶
- 💻 只在單一裝置使用的用戶
- 🚀 臨時記錄、短期使用

- ☁️ 需要跨裝置同步的用戶
- 💾 想要雲端備份的用戶
- 👥 需要分享存檔的用戶
- 🔄 長期使用、重度玩家
